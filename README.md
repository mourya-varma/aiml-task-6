AI & ML INTERNSHIP – TASK 6
Linear Regression – House Price Prediction

Objective:
To build a Linear Regression model to predict house prices and evaluate its performance using regression metrics.

Dataset:
California Housing Dataset (loaded using scikit-learn)

Tools Used:
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

Steps Performed:
1. Loaded the California Housing dataset and converted it into a Pandas DataFrame.
2. Inspected the dataset to understand features and target values.
3. Separated input features (X) and target variable (y).
4. Split the data into training and testing sets using an 80-20 ratio.
5. Trained a Linear Regression model using training data.
6. Predicted house prices on test data.
7. Evaluated the model using:
   - Mean Absolute Error (MAE)
   - Root Mean Squared Error (RMSE)
8. Plotted Actual vs Predicted house prices using a scatter plot.

Model Used:
Linear Regression

Evaluation Metrics:
- MAE: Measures the average absolute error between actual and predicted values.
- RMSE: Measures prediction error magnitude and penalizes large errors.

Files Included:
- task6_linear_regression.py
- README.txt

Outcome:
This task provided hands-on experience in building a regression model, evaluating prediction errors, and visualizing model performance.

Final Result:
Successfully implemented a Linear Regression model for house price prediction and evaluated it using MAE and RMSE.
